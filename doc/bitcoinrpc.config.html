bitcoinrpc.config — Utilities for reading bitcoin configuration files¶

Utilities for reading bitcoin configuration files.

bitcoinrpc.config.read_config_file(filename)¶

Read a simple '='-delimited config file. Raises IOError if unable to open file, or ValueError if an parse error occurs.

bitcoinrpc.config.read_default_config(filename=None)¶

Read bitcoin default configuration from the current user’s home directory.

Arguments:

  • filename: Path to a configuration file in a non-standard location (optional)
